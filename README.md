# smartthings-exporter

a simple app to help monitor your smart devices via prometheus

## Configuration

| Environment Var | Description                    |
|-----------------|--------------------------------|
| `STE_API_TOKEN` | your api token                 |
| `STE_PORT`      | server port (defaults to 9119) |
